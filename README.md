# Unemployment Analysis with Python

## Project Overview

This project analyzes unemployment trends using Python.

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Analysis Performed

- Data Cleaning
- Exploratory Data Analysis
- Unemployment Trend Analysis
- Covid-19 Impact Analysis
- Seasonal Trend Analysis
- State-wise Unemployment Analysis

## Dataset

Unemployment dataset containing unemployment rate,
employment data, labour participation rate, and region information.

## Conclusion

The project identifies unemployment patterns and the impact
of Covid-19 on employment conditions.
